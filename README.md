# 人生重开模拟器

## 游戏简介
一款文字冒险游戏，玩家随机获得初始属性和天赋，体验不同的人生轨迹。

## 核心玩法
- 🎲 随机初始属性（颜值、智力、体质、家境）
- ⭐ 天赋系统（影响人生走向）
- 📖 人生事件（随机触发，影响属性）
- 🎭 多结局（普通/稀有/传说结局）

## 技术栈
- H5：原生JavaScript + Tailwind CSS
- 小程序：微信小程序原生
- 后端：Node.js（可选，用于存档、排行榜）

## 项目结构
```
life-restart/
├── h5/              # H5版本
├── miniprogram/     # 微信小程序版本
├── server/          # 后端服务（可选）
└── assets/          # 资源文件
```
